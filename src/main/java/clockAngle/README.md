Домашнее задание: Угол между стрелками часов
Описание Задачи
В этом домашнем задании вам предстоит разработать Java-программу, которая вычисляет наименьший угол между часовой и минутной стрелками на стандартном аналоговом циферблате. Это задание поможет вам углубить понимание работы с математическими операторами, условными конструкциями (if-else), а также применением логики для решения практических задач.

Основная функциональность

Программа должна запросить у пользователя текущее время в формате часов и минут (например, 3 часа 30 минут).
Необходимо вычислить положение (угол в градусах) часовой стрелки относительно 12 часов.
Необходимо вычислить положение (угол в градусах) минутной стрелки относительно 12 часов.
Рассчитайте абсолютную разницу между этими двумя углами.
Выведите полученный наименьший угол на экран.
Дополнительные условия / Ограничения

Используйте только стандартные библиотеки Java.
Все расчеты должны выполняться в памяти.
Программа должна корректно обрабатывать время, например, 14 часов 55 минут, 3 часа 12 минут и т.д.
Примеры использования

Пример:
Пользователь запускает приложение.
Приложение запрашивает: "Введите часы:"
Пользователь вводит 12.
Приложение запрашивает: "Введите минуты:"
Пользователь вводит 15.
Приложение выводит:
Первый угол: 7.5°
Второй угол: 90°
Наименьший угол между стрелками: 82.5°
Приложение завершает работу.
Входные / Выходные данные
Входные данные:

Два целых числа: часы и минуты
Выходные данные:

Строка в формате: "Наименьший угол между стрелками: X градусов", где X — вычисленное значение угла с плавающей точкой.
Сообщение об ошибке, если ввод некорректен.
Технические требования
Среда выполнения: Консоль
