# Домашнее задание: Угадай номер буквы в ASCII ## **Описание Задачи** Это домашнее задание поможет вам познакомиться с кодировкой **ASCII**, научиться работать с символами (`char`) и их числовыми представлениями, а также отработать обработку ошибок ввода (`try-catch`). Вам предстоит создать простую консольную игру, в которой пользователь должен будет угадать номер в таблице ASCII для заданной буквы. * **Основная функциональность** * Программа должна случайным образом выбрать одну **заглавную латинскую букву** (от 'A' до 'Z'). * Выведите выбранную букву на экран и попросите пользователя угадать её числовой код в таблице ASCII. * Пользователь вводит число. * Программа должна проверить, правильно ли пользователь угадал: * Если ответ правильный, выведите сообщение "Поздравляем\! Вы угадали\!" * Если ответ неправильный, выведите сообщение "Неправильно. Правильный ответ: [правильный ASCII-код]." * В конце игры программа должна завершиться. * **Дополнительные условия / Ограничения** * Используйте только стандартные библиотеки Java. * Для генерации случайных чисел используйте класс `java.util.Random`. * Для считывания ввода пользователя используйте `java.util.Scanner` или `java.io.BufferedReader`. * Реализуйте обработку **некорректного ввода**: * Если пользователь введет нечисловое значение, программа должна вывести сообщение об ошибке (например, "Ошибка: Введите число.") и корректно обработать это исключение (`InputMismatchException` или `NumberFormatException`). * Используйте `try-catch` блоки для обработки возможных исключений. * **Запрещено** использовать готовые методы для получения ASCII-кода символа напрямую, кроме как для сравнения (т.е. нельзя просто `(int) 'A'`). Цель задания — понять, как происходит это преобразование. Однако, вы можете использовать знания о том, что ASCII-коды букв идут последовательно. * Подсказка: ASCII-код символа 'A' равен 65. Вы можете использовать это как отправную точку. * **Примеры использования** * **Пример сценария 1 (Правильный ответ):** 1. *Программа случайно выбрала букву 'C'* 2. Приложение выводит: "Угадайте ASCII-код для буквы 'C':" 3. Пользователь вводит `67`. 4. Приложение выводит: ``` Поздравляем! Вы угадали! ``` 5. Приложение завершает работу. * **Пример сценария 2 (Неправильный ответ):** 1. *Программа случайно выбрала букву 'F'* 2. Приложение выводит: "Угадайте ASCII-код для буквы 'F':" 3. Пользователь вводит `70`. 4. Приложение выводит: ``` Неправильно. Правильный ответ: 70. ``` 5. Приложение завершает работу. * **Пример сценария 3 (Некорректный ввод):** 1. *Программа случайно выбрала букву 'K'* 2. Приложение выводит: "Угадайте ASCII-код для буквы 'K':" 3. Пользователь вводит `привет`. 4. Приложение выводит: ``` Ошибка: Введите число. ``` 5. Приложение завершает работу. ----- ## Входные / Выходные данные **Входные данные:** * Целое число, введенное пользователем. **Выходные данные:** * Запрос на ввод числа. * Сообщение о правильности/неправильности ответа с указанием правильного ASCII-кода. * Сообщение об ошибке в случае некорректного ввода. ----- ## Технические требования * Используйте `java.util.Random` для выбора случайной буквы. Чтобы получить случайную заглавную букву, можно сгенерировать случайное число в диапазоне от 0 до 25, а затем прибавить его к ASCII-коду символа 'A'. * Используйте `java.util.Scanner` или `java.io.BufferedReader` для считывания ввода пользователя. * Применяйте `try-catch` для обработки исключений, возникающих при попытке преобразовать некорректный ввод в число. * **Среда выполнения:** Консоль ----- ## **Бонусные Задания (Необязательно)** * **Несколько попыток:** * Дайте пользователю несколько (например, 3) попыток на угадывание. После каждой неправильной попытки сообщайте, больше или меньше был введенный код по сравнению с правильным. * **Диапазон букв:** * Позвольте пользователю выбирать, какой диапазон букв использовать (например, "только гласные" или "все буквы" включая строчные и заглавные). Это потребует расширения логики генерации случайной буквы и проверки.
