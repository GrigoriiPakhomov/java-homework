Описание Задачи

Реализуй простое консольное приложение для управления складом. Приложение должно позволять пользователю добавлять, просматривать, удалять и изменять товары. Все данные должны храниться в памяти.

Основная функциональность

Добавление товара: Пользователь вводит название, количество и цену товара. Приложение добавляет этот товар в "склад".
Просмотр товаров: Пользователь может просмотреть список всех товаров на складе, включая их названия, количество и цену.
Поиск товара: Пользователь может найти товар по его названию.
Удаление товара: Пользователь может удалить товар со склада по его названию.
Изменение товара: Пользователь может изменить количество и/или цену товара по его названию.
Выход: Предоставь возможность выйти из приложения.
Дополнительные условия / Ограничения

Не использовать внешние библиотеки, кроме стандартных Java.
Все данные должны храниться в памяти.
Код должен быть хорошо структурирован, разбит на классы
