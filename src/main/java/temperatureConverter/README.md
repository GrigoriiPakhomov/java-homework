# Домашнее задание: Конвертер температуры ## **Описание Задачи** Это домашнее задание поможет вам попрактиковаться в использовании условных операторов (`if-else`), работе с математическими операциями и вводом/выводом данных в Java. Вам предстоит создать простую консольную программу, которая будет конвертировать температуру из одной шкалы в другую. * **Основная функциональность** * Программа должна запросить у пользователя **исходную единицу измерения температуры** (например, Цельсий 'C', Фаренгейт 'F' или Кельвин 'K'). * Затем программа должна запросить **значение температуры** в этой единице измерения. * После этого программа должна запросить **единицу измерения, в которую нужно конвертировать температуру** (также ' C', 'F' или 'K'). * На основе введенных данных, программа должна выполнить соответствующее преобразование температуры. * Вывести результат конвертации на экран, округлив его до двух знаков после запятой. * **Дополнительные условия / Ограничения** * Используйте только стандартные библиотеки Java (в частности, `java.util.Scanner` для ввода). * Все вычисления должны выполняться с использованием чисел с плавающей точкой (`double`). * Реализуйте обработку **некорректного ввода**: * Если пользователь ввел некорректную единицу измерения (не 'C', 'F', 'K'). * Если пользователь ввел нечисловое значение температуры. * В этих случаях программа должна вывести соответствующее сообщение об ошибке и завершить работу. * **Примеры использования** * **Пример сценария:** 1. Пользователь запускает приложение. 2. Приложение запрашивает: "Введите исходную единицу измерения (C, F, K):" 3. Пользователь вводит `C`. 4. Приложение запрашивает: "Введите температуру:" 5. Пользователь вводит `25`. 6. Приложение запрашивает: "Введите единицу измерения для конвертации (C, F, K):" 7. Пользователь вводит `F`. 8. Приложение выводит: ``` 25.00 Цельсия = 77.00 Фаренгейта ``` 9. Приложение завершает работу. ----- ## Входные / Выходные данные **Входные данные:** * **Единицы измерения:** Один символ (`C`, `F` или `K`). * **Значение температуры:** Число с плавающей точкой. **Выходные данные:** * Строка, показывающая исходное значение и единицу измерения, а также сконвертированное значение с целевой единицей измерения, округленное до двух знаков после запятой (например, "25.00 Цельсия = 77.00 Фаренгейта"). * Сообщения об ошибках при некорректном вводе. ----- ## Технические требования * **Среда выполнения:** Консоль ----- ## **Бонусные Задания (Необязательно)** * **Проверка на абсолютный ноль:** * Добавьте проверку на температуры ниже абсолютного нуля.
