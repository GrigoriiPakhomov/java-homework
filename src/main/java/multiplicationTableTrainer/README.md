Домашнее задание (Большой Проект): Тренажер Таблицы Умножения
Описание Задачи

Этот проект является полноценным консольным приложением, которое поможет школьникам (или тем, кто хочет освежить память) тренировать таблицу умножения, а также позволит им распечатать её в формате, напоминающем старые тетрадные обложки. Вы будете использовать все полученные ранее знания, включая методы (функции), циклы, условные операторы, * работу со строками и числами*, обработку исключений и формирование JAR-файла для запуска приложения.

Основная функциональность

Главное меню: Программа должна запускаться с интерактивным меню, предлагающим пользователю выбрать одно из действий:
Вывести таблицу умножения (полную)
Запустить режим тренировки
Выход
Режим "Вывести таблицу умножения":
Программа должна вывести на экран полную таблицу умножения от 1*1 до 10*10.
Форматирование: Вывод должен быть аккуратным и выровненным, чтобы результаты выглядели как в таблице (см. примеры). Все числа должны быть выровнены по правому краю, чтобы столбики выглядели ровно.
Возвращение в главное меню после вывода.
Режим "Тренажер":
Компьютер случайным образом генерирует примеры на умножение (например, $7 \times 8$). Числа должны быть от $2$ до $9$ (или до $10$, на выбор).
Программа запрашивает у пользователя ответ.
После каждого ответа программа сообщает, правильный он или неправильный.
Подсчет статистики: Программа должна отслеживать количество правильных и неправильных ответов за одну сессию тренировки.
Пользователь может продолжать тренировку, пока не решит выйти из этого режима.
После выхода из режима тренировки программа должна вывести итоговую статистику за сессию (например, "Всего примеров: 10, Правильных: 8, Неправильных: 2") и вернуться в главное меню.
Дополнительные условия / Ограничения

Структура кода: Программа должна быть хорошо структурирована с использованием методов для каждой логической части
Генерация случайных чисел: Используйте

Непрерывность работы: Приложение должно работать до тех пор, пока пользователь явно не выберет "Выход" из главного меню.
