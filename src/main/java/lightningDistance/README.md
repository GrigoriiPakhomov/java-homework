Домашнее задание: Как далеко ударила молния? ⛈️
Описание Задачи
Вам предстоит написать простую программу на Java, которая определяет расстояние до удара молнии.

Как это работает? Свет движется почти мгновенно, а звук — гораздо медленнее. Поэтому мы сначала видим вспышку молнии и только через некоторое время слышим раскат грома. Зная время между этими двумя событиями, можно вычислить расстояние. Ваша задача — автоматизировать этот расчёт.

Основная функциональность

Программа должна запросить у пользователя количество секунд, прошедшее между вспышкой молнии и звуком грома.
Используя введённое время, программа должна рассчитать расстояние до молнии.
Результат расчёта необходимо вывести в консоль в понятном для пользователя виде.

Примеры использования

Пример сценария:
Пользователь запускает программу.
Программа выводит сообщение: Введите количество секунд между вспышкой молнии и громом:
Пользователь вводит 5 и нажимает Enter.
Программа выводит результат.
Входные / Выходные данные
Вход:

Одно число (целое или дробное), которое представляет время в секундах.
Выход:

Понятное текстовое сообщение с результатом расчёта.
От вспышки до звука прошло % секунд
Молния ударила примерно в 1715 метрах от вас.
Технические требования
Среда выполнения: Консоль.
Бонусные Задания (Необязательно)
Улучшенный вывод: Выведите результат не только в метрах, но и в километрах. Например: Молния ударила примерно в 1715 метрах (1.72 км) от вас.
