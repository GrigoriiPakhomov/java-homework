Домашнее задание: Выводим sin, cos, tan угла
Описание Задачи
Это домашнее задание поможет вам попрактиковаться в использовании математических функций в Java и работе с вводом/выводом данных. Вам предстоит написать программу, которая будет запрашивать у пользователя значение угла в градусах, а затем вычислять и выводить его синус, косинус, тангенс.

Основная функциональность

Программа должна запросить у пользователя значение угла в градусах.
Конвертировать градусы в радианы, так как математические функции Java работают с радианами.
Вычислить синус, косинус, тангенс введенного угла.
Вывести каждое из полученных значений на экран, округлив до двух знаков после запятой.
Особые случаи для тангенса:
Если тангенс не определен программа должна вывести причину
Примеры использования

Пример:

Пользователь запускает приложение.
Приложение запрашивает: "Введите угол в градусах:"
Пользователь вводит 45.
Приложение выводит:
Угол: 45.0 градусов
Синус: 0.71
Косинус: 0.71
Тангенс: 1.00
Приложение завершает работу.
Входные / Выходные данные
Входные данные:

Одно число с плавающей точкой, представляющее угол в градусах.
Выходные данные:

Значения синуса, косинуса, тангенса округленные до двух знаков после запятой.
Сообщения "Тангенс не определен" в соответствующих случаях.
Сообщение об ошибке, если ввод некорректен.
Технические требования
Используйте java.util.Scanner для получения ввода от пользователя.
Используйте Math библиотеку.
Среда выполнения: Консоль
Бонусные Задания (Необязательно)
Котангенс:
Добавьте поддержку котангенса.
Обработка углов вне диапазона 0-360:
Модифицировать программу так, чтобы она корректно обрабатывала углы, выходящие за пределы $0-360$ градусов ( например, $-45$ градусов или $405$ градусов). Это можно сделать, приведя угол к диапазону $0-360$ с помощью операции % (остаток от деления).
