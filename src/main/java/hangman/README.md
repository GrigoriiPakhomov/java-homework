Задание: Игра "Виселица" с использованием ООП
Описание Задачи

Твоя задача — реализовать консольную версию классической игры "Виселица" (Hangman). Игра должна быть полностью управляемой через консоль и демонстрировать правильное использование принципов объектно-ориентированного программирования.

Основная функциональность

Выбор слова: В начале каждой игры приложение должно случайным образом выбирать слово из заранее подготовленного списка.
Отображение прогресса: Вместо угадываемого слова должны отображаться прочерки (
text
Копировать
Полноэкранный режим
_
), соответствующие каждой букве. По мере угадывания букв, прочерки должны заменяться на угаданные буквы.
Количество попыток: У игрока есть ограниченное количество попыток (например, 7). Каждая неверно угаданная буква уменьшает количество оставшихся попыток.
Управление вводом: Приложение должно принимать ввод пользователя (одну букву) и проверять, есть ли такая буква в загаданном слове.
Проверка на победу/поражение: Игра должна заканчиваться, когда игрок угадал все буквы слова (победа) или исчерпал все попытки (поражение).
Список использованных букв: Приложение должно отслеживать и выводить список уже названных букв, чтобы игрок не повторялся.
