# Домашнее задание: Найти максимум/минимум без `Math.*` ## **Описание Задачи** Это домашнее задание поможет вам глубоко понять работу условных операторов `if-else` в Java, научиться самостоятельно реализовывать базовые математические операции и обрабатывать ввод данных. Вам предстоит создать консольную программу, которая будет запрашивать у пользователя три целых числа, а затем определять и выводить наибольшее и наименьшее из них, не используя при этом методов из класса `java.lang.Math` (таких как `Math.max()` или `Math.min()`). * **Основная функциональность** * Программа должна запросить у пользователя первое целое число. * Программа должна запросить у пользователя второе целое число. * Программа должна запросить у пользователя третье целое число. * Используя только операторы сравнения (`>`, `<`, `>=`, `<=`, `==`) и условные конструкции (`if-else`), определите, какое из двух чисел является **максимальным**. * Используя только операторы сравнения и условные конструкции, определите, какое из двух чисел является * *минимальным**. * Выведите найденные максимальное и минимальное значения на экран. * **Дополнительные условия / Ограничения** * **Строго запрещено использовать любые методы из класса `java.lang.Math`**. Цель задания — реализовать логику самостоятельно. * Реализуйте обработку **некорректного ввода**: если пользователь ввел нечисловое значение, программа должна вывести соответствующее сообщение об ошибке и завершить работу. * **Примеры использования** * **Пример сценария 1 (Обычный ввод):** 1. Пользователь запускает приложение. 2. Приложение запрашивает: "Введите первое целое число:" 3. Пользователь вводит `15`. 4. Приложение запрашивает: "Введите второе целое число:" 5. Пользователь вводит `7`. 6. Приложение запрашивает: "Введите третье целое число:" 7. Пользователь вводит `-1`. 8. Приложение выводит: ``` Максимум: 15 Минимум: -1 ``` 9. Приложение завершает работу. ----- ## Входные / Выходные данные **Входные данные:** * Два **целых числа**. **Выходные данные:** * Две строки: "Максимум: X" и "Минимум: Y", где X и Y — найденные значения. * Сообщение об ошибке, если ввод некорректен. ----- ## Технические требования * **Среда выполнения:** Консоль
