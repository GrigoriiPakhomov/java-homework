# Домашнее задание: Длина видео в секундах ## **Описание Задачи** Это домашнее задание поможет вам поработать со **строками**, научиться их разбирать, конвертировать данные и обрабатывать возможные ошибки с помощью **исключений (`try-catch`)**. Вам предстоит написать программу, которая будет считывать продолжительность видео в формате "mm:ss" (минуты:секунды) и преобразовывать её в общую длительность в секундах. * **Основная функциональность** * Программа должна запросить у пользователя строку, содержащую длительность видео в формате "mm:ss" (например, "05: 30", "12:00", "00:45"). * Необходимо **разобрать введенную строку**, чтобы отделить минуты от секунд. * Преобразовать полученные строковые значения минут и секунд в **числовой формат** (целые числа). * Вывести общую длительность в секундах на экран. * **Дополнительные условия / Ограничения** * Используйте только стандартные библиотеки Java. * Для чтения строки используйте `java.util.Scanner` или `java.io.BufferedReader`. * Для разделения строки на части можно использовать методы класса `String` ( например, `split()`, `indexOf()`, `substring()`) или `java.util.StringTokenizer`. * Используйте **`try-catch` блоки** для обработки возможных ошибок ввода: * `NumberFormatException`: Если минуты или секунды не могут быть преобразованы в число (например, "12:zz"). * `ArrayIndexOutOfBoundsException`: Если формат строки некорректен (например, "1244" без двоеточия). * Дополнительно, если захотите, можно обрабатывать общие исключения `Exception` для более широкого перехвата ошибок. * Также необходимо учесть и обработать следующие **логические ошибки**: * Количество секунд должно быть в диапазоне от 0 до 59. Если секунд больше 59, выведите сообщение об ошибке. * Длительность минут и секунд не должна быть отрицательной. * **Примеры использования** * **Пример сценария 1 (Корректный ввод):** 1. Пользователь запускает приложение. 2. Приложение запрашивает: "Введите длительность видео в формате mm:ss:" 3. Пользователь вводит `12:44`. 4. Приложение выводит: ``` Общая длительность видео: 764 секунды. ``` 5. Приложение завершает работу. * **Пример сценария 2 (Секунды больше 59):** 1. Пользователь запускает приложение. 2. Приложение запрашивает: "Введите длительность видео в формате mm:ss:" 3. Пользователь вводит `01:65`. 4. Приложение выводит: ``` Ошибка: Количество секунд не может быть больше 59. ``` 5. Приложение завершает работу. ----- ## Входные / Выходные данные **Входные данные:** * Строка в формате "mm:ss". **Выходные данные:** * Строка в формате: "Общая длительность видео: X секунд.", где X — число секунд. ----- ## Технические требования * Для парсинга строки рекомендуется использовать `String.split(":")` или `StringTokenizer`. * Обязательно используйте **`try-catch`** для перехвата исключений, таких как `NumberFormatException` и `ArrayIndexOutOfBoundsException`. * **Среда выполнения:** Консоль ----- ## **Бонусные Задания (Необязательно)** * **Расширенный формат:** * Поддержка формата "hh:mm:ss" (часы:минуты:секунды). Программа должна корректно обрабатывать оба формата, определяя, какой из них был введен.
