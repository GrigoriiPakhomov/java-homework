# Домашнее задание: Определяем четверть координатной плоскости ## **Описание Задачи** Это домашнее задание поможет вам попрактиковаться в использовании условных операторов (`if-else`) и обработке ввода пользователя в Java. Вам предстоит создать программу, которая будет определять, в какой четверти координатной плоскости находится точка, заданная своими координатами (X и Y). * **Основная функциональность** * Программа должна запросить у пользователя значение **координаты X** (целое число). * Программа должна запросить у пользователя значение **координаты Y** (целое число). * Используя условные операторы (`if-else`), определите, в какой из четырех четвертей находится точка * Выведите результат на экран. * **Примеры использования** * **Пример:** 1. Пользователь запускает приложение. 2. Приложение запрашивает: "Введите координату X:" 3. Пользователь вводит `5`. 4. Приложение запрашивает: "Введите координату Y:" 5. Пользователь вводит `10`. 6. Приложение выводит: ``` Точка (5, 10) находится в I четверти. ``` 7. Приложение завершает работу. ----- ## Входные / Выходные данные **Входные данные:** * Два **целых числа**: координата X и координата Y. **Выходные данные:** * Строка, указывающая, в какой четверти находится точка (например, "Точка (X, Y) находится в III четверти." ----- ## Технические требования * **Среда выполнения:** Консоль
