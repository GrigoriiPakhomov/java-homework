Домашнее задание: Сломанная строка
Описание Задачи

Это домашнее задание поможет вам глубоко поработать со строками, научиться их разбирать, модифицировать и собирать обратно. Вам предстоит написать программу, которая будет "исправлять" перепутанные строки, где каждая пара символов поменялась местами.

Представьте, что у вас есть строка, например, "псаривьт". Если рассмотреть её по парам символов, то пс превратилось в сп, ар в ра и так далее -> справить. Ваша задача — написать функцию, которая вернёт строку в исходное состояние, поменяв местами каждую последовательную пару символов.

Основная функциональность

Программа должна запросить у пользователя строку, которую нужно "исправить".
Если длина строки нечетная, последний символ остаётся на своём месте. Например, для строки "АБВ" результатом будет "БАВ".
Ваша программа должна пройти по строке, меняя местами каждый символ с соседним.
Первый символ меняется со вторым.
Третий с четвёртым.
Пятый с шестым и так далее.
Сформированную исправленную строку выведите на экран.
Примеры использования

Пример:

Пользователь запускает приложение.
Приложение запрашивает: "Введите сломанную строку:"
Пользователь вводит оПомигет сирпватиь тсорик!.
Приложение выводит:
Исправленная строка: Помогите исправить строки!
Приложение завершает работу.
Входные / Выходные данные

Входные данные:

Одна строка (может содержать любые символы, включая пробелы и знаки препинания).
Выходные данные:

Строка, представляющая исправленный вариант введенной строки.
Технические требования

Среда выполнения: Консоль
Бонусные Задания (Необязательно)

Генератор сломанных строк:
Напишите функцию, которая делает обратное действие: берёт обычную строку и "ломает" её, меняя местами каждые два символа.
Позвольте пользователю выбирать количество символов в подстроке для их смещения
