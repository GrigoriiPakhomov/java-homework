Домашнее задание: Примитивный калькулятор ## **Описание Задачи** Это домашнее задание поможет вам закрепить навыки работы с вводом/выводом данных, различными типами переменных, математическими операциями и условными конструкциями (`if-else` или `switch`) в Java. Вам предстоит создать простой консольный калькулятор, который выполняет четыре основные арифметические операции: сложение, вычитание, умножение и деление. * **Основная функциональность** * Программа должна запросить у пользователя первое число (с плавающей точкой). * Программа должна запросить у пользователя второе число (с плавающей точкой). * Программа должна запросить у пользователя **операцию**, которую нужно выполнить (например, `+`, `-`, `*`, `/`). * На основе выбранной операции, выполните соответствующее арифметическое действие. * Выведите **результат** вычисления на экран, округлив его до двух знаков после запятой. * **Особый случай для деления:** Если пользователь попытается разделить на ноль, программа должна вывести сообщение "Ошибка: Деление на ноль невозможно." и завершить работу. * **Дополнительные условия / Ограничения** * Используйте только стандартные библиотеки Java (в частности, `java.util.Scanner` для ввода). * Реализуйте обработку **некорректного ввода** и **некорректной математики** * **Примеры использования** * **Пример:** 1. Пользователь запускает приложение. 2. Приложение запрашивает: "Введите первое число:" 3. Пользователь вводит `10.5`. 4. Приложение запрашивает: "Введите операцию (+, -, \*, /):" 5. Пользователь вводит `+`. 6. Приложение запрашивает: "Введите второе число:" 7. Пользователь вводит `5.2`. 8. Приложение выводит: ``` Результат: 15.70 ``` 9. Приложение завершает работу. ----- ## Входные / Выходные данные **Входные данные:** * Первое число (тип `double`). * Символ операции (тип `String` или `char`). * Второе число (тип `double`). **Выходные данные:** * Строка в формате: "Результат: X.YY", где X.YY — вычисленное значение, округленное до двух знаков после запятой. * Сообщения об ошибках при некорректном вводе или делении на ноль. ----- ## Технические требования * Для выбора арифметической операции используйте **`switch`**. * **Среда выполнения:** Консоль ----- ## **Бонусные Задания (Необязательно)** * **Расширенный функционал:** * Добавить возможность возведения в степень (оператор `^`). * Добавить возможность вычисления квадратного корня из первого числа (например, операция `sqrt`). При этом второе число может не запрашиваться или игнорироваться.
