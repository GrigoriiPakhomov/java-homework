# Домашнее задание: Является ли число степенью какого-то числа?

## Описание задачи

В этом домашнем задании вам предстоит написать программу на Java, которая будет определять, является ли введённое пользователем целое число **степенью другого целого числа**.

Например:

* 8 является степенью 2 (2^3),
* 9 — степенью 3 (3^2).

Число 12 не является степенью никакого целого числа, кроме `12^1`.

Мы будем рассматривать только степени, где:

* основание `b > 1`
* показатель степени `p > 1`

---

## Основная функциональность

* Программа должна запросить у пользователя **целое положительное число `N`** (исследуемое число).
* Программа должна определить, существуют ли такие целые числа `b` и `p`, что `b^p = N`.
* Если такое представление найдено, вывести сообщение вида:

```
Число N является степенью числа b (b^p)
```

Например:

```
Число 8 является степенью числа 2 (2^3)
```

---

## Особые случаи

* Отрицательные числа — программа должна работать только с положительными целыми числами.
* Число `1` не учитывается.

---

## Дополнительные условия / ограничения

* Не использовать `Math`.

---

## Примеры использования

### Пример сценария 1 (N = 8)

1. Пользователь запускает приложение.
2. Приложение запрашивает:

```
Введите целое положительное число N:
```

3. Пользователь вводит:

```
8
```

4. Приложение выводит:

```
Число 8 является степенью числа 2 (2^3)
```

5. Приложение завершает работу.

---

## Входные / выходные данные

### Входные данные

* Одно целое положительное число `N`.

### Выходные данные

* Сообщение о том, является ли число `N` степенью другого числа, с указанием основания и показателя, если это так.
* Сообщение об ошибке при некорректном вводе.
