# Java Homework Project

Репозиторий для домашних заданий по подготовке к позиции Java Developer.

## Структура
- `src/` - исходный код
- Каждая задача в отдельном пакете

## Темы
1. Java Core
2. ООП и паттерны
3. Многопоточность
4. Spring Framework

## Как использовать
1. Каждую неделю/тему создавать отдельную папку в `src/`
2. Коммитить изменения через IntelliJ IDEA
3. Создавать Pull Request для ревью
